## 🚀 核心功能亮点

### 1. 解决用户适配性问题
- **问题**：硬编码模型参数导致Aider启动黑屏
- **解决方案**：交互式模型配置，用户可选择适合的模型
- **支持模型**：deepseek-chat、deepseek/deepseek-chat、gpt-4、claude-3等

### 2. 简化使用流程
- **传统方式**：编辑配置文件 → 启动工具 → 启动监控
- **新方式**：运行 `python companion_launcher.py` → 按提示操作 → 自动完成所有步骤

### 3. 智能权限管理
- **四种模式**：ask/auto/auto_approve/silent
- **自动响应**：根据白名单自动决策
- **实时通知**：权限请求时发送通知

### 4. 多工具支持
- **已支持**：Aider、Claude Code、Cursor、Codex、Tabnine
- **扩展性**：模块化设计易于添加新工具

## 📋 使用方式总结

### 基本命令
```bash
# 交互式启动（推荐）
python companion_launcher.py

# 快速启动
python companion_launcher.py quick aider .

# 状态管理
python companion_launcher.py status
python companion_launcher.py stop
python companion_launcher.py help
```

### 交互式流程
1. **检测工具** → 2. **选择工具** → 3. **配置模型** → 4. **设置权限** → 5. **启动监控**

### 配置示例
```yaml
# 自动生成的配置
projects:
  - name: "Aider监控"
    tool: "aider"
    terminal_config:
      command: "aider"
      args: ["--model", "deepseek/deepseek-chat"]
      auto_response:
        enabled: true
        allow_response: "\n"  # 回车作为yes
```
